# AppFacturas

Aplicación Java de consola para la gestión básica de facturas y productos.

## Funcionalidades

- Creación de facturas
- Gestión de productos
- Cálculo de subtotal e IVA
- Cálculo de total final
- Formato personalizado de fecha

## Tecnologías utilizadas

- Java
- Git
- GitHub

## Objetivo del proyecto

Este proyecto fue utilizado como práctica para aprender y aplicar flujo profesional de Git y GitHub:

- Uso de ramas feature
- Commits semánticos
- Pull Requests
- Merge de ramas
- Resolución de conflictos
